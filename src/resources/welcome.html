Welcome to the Seattle Weather Service

This site does not have any interactive functionality. You can query the weather data from the command line with. Here are some example commands

See all weather data

        curl localhost:3000/query

See a limited number of weather data

        curl localhost:3000/query?limit=7

See only sunny days

        curl localhost:3000/query?weather=sun

See only a specific day

        curl localhost:3000/query?date=2012-11-30

See a limited number of days with rain

        curl localhost:3000/query?weather=rain&limit=10
